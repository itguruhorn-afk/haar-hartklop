## Home Page

Required sections:

- rotating hero images with text overlay
- floating or sticky "Join the sisterhood" call-to-action
- word from founder
- mission / vision
- Bible study testimonials
- shop preview
- weekly encouragement / YouTube podcast link
- social links

Hero requirement:

- Use a photo-based hero, not a generic gradient.
- Allow multiple rotating photos.
- Admin must be able to change photo, heading, subheading and button links.

Founder section:

- Layout should support one side image and one side text.
- Admin must be able to update photo, text and section background color.

Testimonials:

- Use carousel/slider behavior with arrows.
- Admin must be able to update quote, author, optional photo and display order.

## Virtual Bible Study Page

Required sections:

- photo hero with text overlay
- summary of what someone receives when they subscribe
- Bible study material purchase area
- testimonials
- links to socials
- newsletter signup
- Bible study registration/signup

Special notes:

- Limited participant capacity should be supported.
- Paid subscription/registration should be planned.
- Zoom links must not be public. Paid users should receive protected access.
- Consider future community groups for prayer, communication and announcements.

## Events Page

Required sections:

- upcoming events
- past events
- event gallery
- video embed area

Event types:

- meet and greet
- camps
- conference

Admin must be able to add events even if there are none at launch.

## Salvation Page

Form fields:
