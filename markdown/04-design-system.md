# Design System

## Visual Direction

The workbook references a feminine, warm, faith-centered design with:

- rounded photos
- photo-led hero areas
- editable colors and fonts
- soft backgrounds
- elegant typography
- red, nude, purple, light blue and a touch of olive green

The site should feel:

- welcoming
- polished
- modern
- gentle
- ministry/community focused
- not corporate
- not overly decorative

## Colors

Suggested palette:

```text
Deep red: #8F1717
Wine/purple: #58223B
Nude/rose: #D7A893
Soft blush: #FFF8F4
Warm cream: #F4E6DD
Light blue: #BFD6DE
Olive: #7A7B45
Ink: #271718
Muted text: #756667
```

Admin should be able to update primary and accent colors.

## Typography

Workbook font notes:

- Playfair Display
- Montserrat
- Abramo

Recommended web-safe implementation:

- headings: Playfair Display
- body: Montserrat
- optional decorative/script font only for logo or accent text

If Abramo is not licensed or available, do not use it from an unofficial source.

## Layout Guidelines

Use:

- full-width photo hero with text overlay
- clear sections with generous spacing
- rounded image crops
- elegant cards for products/events only
- carousel for testimonials and hero photos
- simple forms with clear labels and consent text

Avoid:

- generic stock-looking gradients
- text-heavy homepage sections
- overly complicated animations
- hard-to-read script fonts for body text

## Responsive Requirements

Mobile:

- stacked layouts
- large readable buttons
- no overlapping hero text
- forms should be easy to complete
- navigation should collapse into a menu

Desktop:

- strong first impression hero
- comfortable reading width
- card grids for shop/events
- admin dashboard should use tables or structured panels

## Accessibility

Include:

- semantic HTML
- visible focus states
- alt text for images
- form labels
- color contrast checks
- keyboard-accessible navigation
- accessible carousel controls
